# assets/

Taruh foto & file musik asli kalian di sini, lalu impor di `src/data/data.js`.

Contoh import gambar di `data.js`:

```js
import foto1 from '../assets/foto1.jpg'
// lalu pakai: { src: foto1, caption: '...' }
```

Selama belum ada aset asli, `data.js` memakai placeholder dari
picsum.photos (gambar) dan soundhelix.com (audio).
